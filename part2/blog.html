What is Angular.js?

February 8, 2016 by Javoni

Angular is a way to extend the HTML language in a way that offers dynamic views. HTML only offers static views. I explain the basics by going through the steps of my most recent Angular application. Follow along with my code in github by clicking on AngularJs(code) in the navigation bar at the top.

After extensive Googling here is my process...

My Process...

First I created a file called tutorialsService.js where I create an app that will take my tutorials.json file and turn all of the json strings into html objects that will be displayed in a table.

I created a service in the app called tutorialsService which is a function that accepts two parameters: $http and $q. $http allows us to call to our tutorials.json file and $q.defer() is stored in a variable called deferred. This means that anything can be stored in the variable deferred so whatever information is stored in that variable can be used later. In this case I store the data from our json file in the deferred variable. All of that is done in this statement: var deferred = $q.defer(); $http.get("tutorials.json").then(function (data) {deferred.resolve(data);});

The next function is called getTutorials and it will return all of the information that is stored in the deferred variable.

The last thing in the tutorialsService.js file is the creation of the controller, tutorialsCtrl. This controller has a function that accepts two parameters, $scope and the all of the information in the tutorialsService app. First the controller calls the promise that the deferred variable made which contains all of the data from the json file and it assigns it to the scope and finally displays the information to the console just to confirm that all of the information is being processed correctly into the controller.

The final part of this angular app is in the body of the table in our index.html form. data-ng-repeat="tutorial in tutorials" loops through each tutorial object in the json file and displays them in the corresponding columns inside the double curly brackets.
